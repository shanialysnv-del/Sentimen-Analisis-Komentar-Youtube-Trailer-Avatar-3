# Sentimen-Analisis-Komentar-Youtube-Trailer-Avatar-3
Repository ini merupakan proyek UAS Analisa Big Data yang mengimplementasikan pipeline analisis sentimen berbasis teks, mulai dari pengumpulan data YouTube, preprocessing, feature extraction, balancing data (SMOTE), pelatihan dan evaluasi model machine learning, hingga implementasi model ke dalam aplikasi prediksi sentimen menggunakan Gradio.
